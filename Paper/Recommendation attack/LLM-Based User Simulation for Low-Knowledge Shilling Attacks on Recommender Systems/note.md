# LLM-Based User Simulation for Low-Knowledge Shilling Attacks on Recommender Systems

## 一句话总结
这篇论文展示了 LLM 不只可以帮推荐系统服务用户，也可以帮攻击者伪装出更真实的恶意用户。

## 论文做了什么
作者研究低知识场景下的 shilling attack，认为传统攻击方法过度依赖内部数据或简单启发式，且很少充分利用文本评论这一攻击渠道。

## 核心方法
论文提出 Agent4SR，用 LLM agent 协同完成目标画像构造、物品选择、评分生成和评论撰写，并结合记忆检索机制让伪造用户行为既有攻击性又保持表面合理性。

## 结果与意义
它说明推荐系统面对的安全风险正在从“伪造评分”升级到“伪造完整用户行为”，对评论型和多模态推荐系统尤其值得警惕。
