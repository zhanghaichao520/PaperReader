# Paragon: Parameter Generation for Controllable Multi-Task Recommendation

## 一句话总结
这篇论文想解决的不是“怎么训练一个固定推荐模型”，而是“需求变了以后怎么快速切模型目标”。

## 论文做了什么
作者面向商业推荐里常见的动态任务需求，比如准确率和多样性偏好的变化，提出一种无需每次都重新训练的 controllable multi-task recommendation 方法。

## 核心方法
Paragon 的思路是根据当前任务要求直接生成推荐模型参数，而不是从头重训模型。这样部署后的模型可以随着控制目标切换而快速适配新的优化方向。

## 结果与意义
论文报告生成参数的方式相比重训练能大幅减少计算时间，同时保持有竞争力的效果，说明“参数生成式适配”很适合在线系统中的快速任务切换。
