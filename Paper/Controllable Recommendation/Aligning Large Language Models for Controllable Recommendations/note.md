# Aligning Large Language Models for Controllable Recommendations

## 一句话总结
这篇论文的重点不是单纯把 LLM 用来做推荐，而是让 LLM 真正学会遵守推荐里的控制指令。

## 论文做了什么
作者指出很多 LLM 推荐方法只追求离线准确率，却经常出现格式错误、重复推荐和不听指令的问题，因此提出专门面向 controllable recommendation 的对齐训练流程。

## 核心方法
方法分为监督学习和强化学习两阶段：先构造推荐、搜索、类别控制、类别比例控制等任务，并用传统推荐模型补充监督信号；再用多种奖励继续做对齐，强化模型的指令遵循能力并减少格式错误。

## 结果与意义
在 Steam 和 Amazon Movie 上，方法能在保持较好准确率的同时显著提升指令跟随能力，说明 LLM 做推荐代理时，“对齐”本身就是核心问题。
