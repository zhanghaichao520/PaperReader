# Editable User Profiles for Controllable Text Recommendations

## 一句话总结
这篇论文试图把原本黑盒的用户画像改成可读、可改、可控的概念集合。

## 论文做了什么
作者关注文本推荐里“用户为什么会被这样表示”这个问题，提出一种基于概念瓶颈的用户画像建模方式，让用户控制推荐时不再只能被动接受潜在向量表示。

## 核心方法
LACE 会从用户读过或交互过的文本里检索出一组人类可读概念，再学习这些概念对用户的个性化权重，并基于这套可编辑概念画像来做推荐，因此用户可以通过修改概念来干预结果。

## 结果与意义
这类方法的价值不只是可解释，也在于提供了真正可操作的控制接口，把“看懂画像”和“改写画像”合并到了一个统一框架里。
