# Taxonomy-Guided Zero-Shot Recommendations with LLMs

## 一句话总结
这篇论文想让 LLM 在零样本推荐里少一点“自由发挥”，多一点由物品 taxonomy 引导的结构化判断。

## 论文做了什么
作者指出零样本 LLM 推荐常受限于上下文长度、物品信息无结构以及生成不受约束，因此提出用类别层级知识系统性地辅助推荐。

## 核心方法
TAXREC 把 taxonomy 作为推荐推理的骨架，用类别信息去组织候选、约束生成并补充物品语义，让 LLM 不必只靠松散自然语言自己推断推荐空间。

## 结果与意义
这类方法的意义在于把现成的业务 taxonomy 重新利用起来，弥补纯 LLM 零样本推荐在结构化先验上的不足。
