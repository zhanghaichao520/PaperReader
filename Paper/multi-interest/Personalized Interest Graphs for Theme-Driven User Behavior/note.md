# Personalized Interest Graphs for Theme-Driven User Behavior

## 一句话总结
这篇论文关注的是跨品类但围绕同一主题的兴趣，希望推荐系统能看见这种“主题型用户画像”。

## 论文做了什么
作者发现很多电商用户的兴趣并不是单品类的，而是围绕某个长期主题在多个类别间展开，因此提出一个基于 LLM 生成用户兴趣图的端到端推荐框架。

## 核心方法
系统先用 LLM chain 近实时生成多层级 user-interest graph，再把图拆成兴趣路径做排序，并把高价值兴趣扩展成检索查询去召回跨类目物品；为控制成本，还采用 teacher-student 方案训练小模型替代大模型。

## 结果与意义
该系统已在 eBay 大规模上线，A/B test 提升了用户对新类别内容的点击、购买和买家数，说明“主题驱动”的跨类别推荐确实能带来更多惊喜感而不只是噱头。
