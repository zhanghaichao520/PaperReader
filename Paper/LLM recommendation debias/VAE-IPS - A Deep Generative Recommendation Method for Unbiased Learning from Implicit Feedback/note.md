# VAE-IPS

这篇论文研究隐式反馈训练中的选择偏置问题，比如热门偏置、位置偏置等。作者指出，虽然 VAE 已经是隐式反馈推荐中的强基线，但现有去偏方法大多没有进入生成式建模框架，因此直接用点击数据训练 VAE 仍然会学到有偏的用户偏好。

论文提出 **VAE-IPS**，核心做法是把 **逆倾向加权（IPS）** 引入 VAE 的训练目标。作者先写出一个理想的、基于真实相关性的生成式目标，再说明如果直接用观测到的点击替代真实相关性，会得到有偏估计；随后用 IPS 对每条观测进行按曝光概率倒数的重加权，构造出对选择偏置无偏的 VAE 训练目标。可以把它理解成：模型结构还是生成式 VAE，但优化目标被改成了“对偏置校正后的重加权似然”，从而让 VAE 在隐式反馈场景下学得更公平、更接近真实偏好。
