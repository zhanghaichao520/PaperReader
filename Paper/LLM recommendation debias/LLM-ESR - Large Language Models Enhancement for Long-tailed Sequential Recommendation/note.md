# LLM-ESR

这篇论文解决长尾序列推荐问题，目标是同时改善长尾物品和长尾用户的推荐效果。作者的基本思路不是直接在线调用大模型生成推荐，而是先离线用 LLM 提取语义信息，再把这些语义表示注入传统序列推荐模型，这样既利用了 LLM 的语义能力，又不会引入额外在线推理成本。

方法上主要有两块。第一，针对 **长尾物品**，作者设计了 **dual-view modeling**，把 LLM 提供的语义表示和传统协同过滤信号结合起来共同建模，并冻结语义嵌入，避免微调时破坏原始语义结构。第二，针对 **长尾用户**，作者提出 **retrieval augmented self-distillation**，先基于 LLM 得到的用户语义表示检索相似用户，再用这些更丰富的行为信息来蒸馏和增强目标用户的偏好表示。整体上，LLM-ESR 是一个“离线语义增强 + 双视角物品建模 + 相似用户蒸馏”的框架。
